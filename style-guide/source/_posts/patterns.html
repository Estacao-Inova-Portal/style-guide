--- title: Padrões (Background patterns) categories: Componentes ---

Padrões de fundo trazem grafismos para o background dos seus elementos

<div class="card pattern pattern--orange-goop util--overflow-hidden">
  ...
</div>

A classe .pattern faz modificações nos pseudo-elementos :before e :after do elemento em que é aplicado, assim garantindo que os elementos de padrões funcionarão.

Em .cards é recomendável adicionar a classe global de utilidade chamada de .util--overflow-hidden, assim ela garante que o desenho do padrão não será pintado para fora do espaço do cartão.

Os modificadores de Padrões de fundo

São os modificadores que trazem os grafismos e padrões.

Modificadores base

Os modificadores base automaticamente deixam o fundo do elemento em que são aplicados branco, independente do elemento em que são aplicados.

Ao todo são 4 modificadores base:

  • .pattern--orange-goop
  • .pattern--gray-goop
  • .pattern--orange-goop-right-side
  • .pattern--gray-goop-right-side
<div class="card pattern pattern--orange-goop util--overflow-hidden">
  ...
</div>
<div class="card pattern pattern--gray-goop util--overflow-hidden">
  ...
</div>
<div class="card pattern pattern--orange-goop-right-side util--overflow-hidden">
  ...
</div>
<div class="card pattern pattern--gray-goop-right-side util--overflow-hidden">
  ...
</div>
