Conjunto de ícones presentes nos aplicativos Unimed Cliente e Unimed Cooperado.

Como usar

Para usar os ícones o primeiro passo é fazer o download deles. Depois você deve colocar os arquivos dentro de img/icons no diretório do seu projeto.

Agora para utilizar os ícones no seu HTML você só precisa referenciar eles, lembrando que eles sempre tem o prefixo ico_, ou seja /img/icons/ico_NOME-DO-ICONE é o caminho que deve ser utilizado.
