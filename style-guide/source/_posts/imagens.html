Classes universais para manipulação de imagens

Imagem no formato de Thumbnail

A classe .img--thumbnail ao ser aplicada em um img força ela a ter um tamanho de thumbnail, no caso desse projeto tamanho máximo de 32 x 32 Unidades básicas, ou seja 157,5px x 157,5px.

  src="/img/illustrations/home/light-weight.svg" 
  class="img--thumbnail" 
  alt="Imagem de balões representando leveza"
/>
<img 
  src="/img/icons/ico_simbolo-medicina.svg"

  alt="Imagem de uma cruz médica"
/>

Imagem centralizada

Com .img--centered aplicado, a imagem será posicionada no centro do seu elemento pai.

<img 
  src="../img/illustrations/home/light-weight.svg" 
  class="img--thumbnail img--centered" 
  alt="Imagem de balões representando leveza"
/>

Ícones clicáveis

A classe .img--clickable-icon cria ícones com tamanho mínimo ideal para área de toque. Esses ícones por padrão se alinham à direita.
