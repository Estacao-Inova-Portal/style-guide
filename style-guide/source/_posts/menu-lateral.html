--- title: Menu lateral categories: Componentes ---

Menus laterais ajudam na navegação, criam acesso aos ítens presentes na sua aplicação.

No canto esquerdo das páginas desse Guia de Estilo se encontra um Menu Lateral, abaixo o código que forma ele:

<aside class="grid__menu grid--display-none-mobile">
  <nav class="menu menu--fixed">
    <h2>Guia de estilos</h2>
    ...
    <details>
      <summary>Introdução</summary>
      <ul class="menu__link-list">
        <li><a href="/introducao/">Introdução</a></li>
        ...
      </ul>
    </details>
    <details>
      <summary>Layout</summary>
      <ul class="menu__link-list">
        <li><a href="/grid/">Grid</a></li>
        ...
      </ul>
    </details>
    ...
  </nav>
</aside>

Elementos do Menu lateral

Lista de links

Quando dividir os links no menu em listas não ordenadas ul, existe uma classe para organizar eles sem os elementos de lista, e com respiro entre eles. Essa classe é a .menu__link-list e deve ser utilizada no ul.

  • Hermione Granger
  • Severus Snape
  • Albus Dumbledore
  • Neville Longbottom
  • Ginny Weasley
<ul class="menu__link-list">
  <li><a href="#">Hermione Granger</a></li>
  <li><a href="#">Severus Snape</a></li>
  <li><a href="#">Albus Dumbledore</a></li>
  <li><a href="#">Neville Longbottom</a></li>
  <li><a href="#">Ginny Weasley</a></li>
</ul>

Modificadores do Menu lateral

Menu Fixo

A classe .menu--fixed deixa o menu preso, de forma fixa, estando sempre aparente na tela do dispositivo do usuário.

<nav class="menu menu--fixed">
  ...
</nav>
