--- title: Polyfill (Internet Explorer) categories: JavaScript ---

Utilizamos JavaScript moderno e atualizado para montar os nossos scripts.

Para funcionar em navegadores antigos - em especial no Internet Explorer - temos dois scripts. Um deles é um Script para validar as capacidades do navegador e, se necessário, adicionar algumas funcionalidades em seu código, e um segundo com o código dos nossos scripts reescritos em JavaScript antigo.

Para funcionar você precisa dos dois arquivos.

Baixar validador (.js) Baixar JS com estilo antigo (.js)

Como funciona?

O primeiro passo é adicionar ambos os arquivos na pasta /js do seu projeto. Caso você utilize uma pasta com outro nome, precisa modificar o caminho dentro dos arquivos.

O segundo passo é referenciar o arquivo polyfill.js (validador) em sua página, o outro arquivo não precisa ser chamado em seu HTML pois, se necessário, será chamado pelo próprio polyfill.

(...)
  <script src="/js/polyfill.js"></script>
(...)
