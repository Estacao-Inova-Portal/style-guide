Formulários e seus elementos permitem capturar contatos, criar campos de busca... São uma das formas mais básica de interação entre uma aplicação e um usuário

Elementos básicos de um formulário

Os elementos mais básicos de formulário são tratados aqui diretamente pela sua tag HTML, sem o uso de classes adicionais. Isso garante acessibilidade e performance, onde os campos utilizados tem exatamente o comportamento esperado pelo navegador.

Precisando de ajuda para montar formulários? A Mozilla tem uma documentação bem completa que pode te ajudar a fazer formulários eficientes e semânticos.

Eu sou um formulário simples 
<form>
  <fieldset>
    <legend>Eu sou um formulário simples</legend>
    <label to="campo">Primeiro campo</label>
    <input id="campo" type="text" placeholder="Insira um dado">
    <label to="outro-campo">Mais um campo aqui</label>
    <input id="outro-campo" type="text" placeholder="Insira algum outro dado">
    <input type="submit" value="Enviar os dados">
  </fieldset>
</form>

Os elementos adicionais de formulários

Dicas de campo

A classe .form__hint pode ser adicionada em um elemento de texto, preferenciamente um span, que deve ser colocado dentro da label. Sua função é adicionar dicas para o usuário em como utilizar algum campo.

<form>
  <label to="senha" placeholder="Insira um dado">Crie uma senha
      <span class="form__hint">Ela deve ter no mínimo 8 caracteres e pelo menos um número e uma letra maiúscula.</span>
    </label>
    <input pattern="/^(?=.*\d)(?=.*[a-z])(?=.*[A-Z])(?=.*[a-zA-Z]).{8,}$/gm" id="senha" type="password">
</form>

Múltiplas escolhas

Uma div com a classe .form__multiple-choice deve ser o elemento pai de cada um dos inputs + label de elementos de múltipla escolha checkbox e radio.

Selecione apenas uma opção: 
<form>
  <legend>Selecione apenas uma opção:</legend>
  <div class="form__multiple-choice">
    <input type="radio" name="radio-input" id="radio-a">
    <label for="radio-a">Doláres</label>
  </div>
  <div class="form__multiple-choice">
    <input type="radio" name="radio-input" id="radio-b">
    <label for="radio-b">Reais</label>
  </div>
  <div class="form__multiple-choice">
    <input type="radio" name="radio-input" id="radio-c">
    <label for="radio-c">Pesos</label>
  </div>
</form>
