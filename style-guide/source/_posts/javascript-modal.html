Usamos JavaScript para tornar os modais interativos

A estrutura do Modal foi construída baseada no projeto a11y-dialog, criado por Hugo Giraudel. O modal foi desenvolvido de forma a garantir a melhor experiência para usuários com necessidades especiais e que acessam a internet por sistemas de ditado.

Criando um modal

Na página do componente modal é explicado qual a semântica que deve ser utilizada para criar um modal.

Para ele se tornar interativo é necessário:

  • Criar um id no elemento que receber a classe .modal__container.
  • No botão/link que for utilizado para abrir o modal adicionar o atributlo data-modal-show. O valor desse atributo deve ser o id do elemento com classe .modal__container
  • Nos botões/links que forem utilizados para fechar o modal, adicionar o atributo data-modal-hide.

Por exemplo, se o elemento container do modal ter o id #modalUnimed, o botão que chama ele deve ter o atributo data-modal-show="modalUnimed".

E... Pronto! O script irá rastrear todos os botões que chamam modais e deixar eles prontos para funcionar.

Modal aberto!

Você abriu um modal, o container dele tem o id #modal.

E esse id é o valor que o parâmetro data-modal-show recebeu no botão Abrir Modal. 

<button type="button" data-modal-show="modal">Abrir modal</button>
<div id="modal" class="modal__container">
  <!-- The overlay -->
  <div tabindex="-1" data-modal-hide class="modal__overlay"></div>

  <!-- The modal -->
  <dialog class="modal card" aria-labelledby="modal-title" role="dialog">
    <!-- Close button -->
    <button type="button" class="modal__close-btn" data-modal-hide aria-label="Fechar a janela de modal">
      &times;
    </button>
    <section class="modal__header">
      <h2 id="modal-title"><strong class="text--gradient-hot-colors">Modal aberto!</strong></h2>
    </section>

    <section class="modal__content">
      <p class="text--big">Você abriu um modal, o container dele tem o id <code>#modal</code>.</p>
      <p>E esse <code>id</code> é o valor que o parâmetro <code>data-modal-show</code> recebeu no botão Abrir Modal.</p>
    </section>
  </dialog>
</div>
