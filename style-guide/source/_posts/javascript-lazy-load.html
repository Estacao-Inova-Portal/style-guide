--- title: Carregamento Dinâmico categories: JavaScript ---

Usamos JavaScript para carregar imagens e vídeos de forma dinâmica (Lazy Load)

Apesar de desde a versão 75 do Chrome, ele aceitar o parâmetro loading="lazy" até o momento ele é o único navegador a fazer carregamento dinâmico de imagens de forma nativa.

Apesar disso a sugestão é sim utilizar esse parâmetro nas suas imagens pois navegadores que não suportarem vão apenas ignorar.

Esse parâmetro deve ser utilizado da seguinte maneira:

<img src="unimed.jpg" loading="lazy" alt="..." />

E isso garantirá o carregamento dinâmico de imagens no Chrome a partir da versão 75.

Carregamento dinâmico universal de imagens e vídeos

Para te ajudar a desenvolver sites com boa performance, dentro dos nossos controles de JavaScript, temos um script que roda automaticamente após o carregamento da página e faz o carregamento de imagens e vídeos de forma dinâmica.

Como fazer o carregamento dinâmico funcionar

Para fazer o carregamento dinâmico funcionar, nas imagens e vídeos que deseja aplicá-lo, troque o parâmetro src por data-src, e pronto, essa mídia irá carregar de forma dinâmica.

<img data-src="unimed.jpg" loading="lazy" alt="..." />

Lembre-se, para o carregamento dinâmico funcionar, a mídia não pode ter o atributo src, apenas data-src ou o carregamento dinâmico se tornará inútil.
