--- title: Menu mobile categories: Componentes ---

É o menu responsável pela navegação em dispositivos móveis.

O Menu mobile por padrão é renderizado em dispositivos móveis (um modificador troca esse comportamento). Para telas maiores a navegação é recomendável pelo Menu Lateral como também pela Barra de navegação.

Ele sempre se encontrará de forma fixa na área inferior da tela do usuário.

<nav class="mobile-nav">
  <a href="javascript:void(0)" class="util--no-style">
    <figure class="mobile-nav__button">
      <img src="../img/icons/ico_autorizacoes.svg" alt="Autorizações">
      <figcaption>Autorizações</figcaption>
    </figure>
  </a>
 ...
</nav>

Os elementos do menu mobile

Botão de ação

São os elementos de botão de navegação. Coloque a classe .mobile-nav__button em uma figure que deve ter como elementos filhos uma img de um ícone e um figcaption com a descrição do botão.

Caso você encaminhe o usuário para uma página, a recomendação é encapsular o elemento .mobile-nav__button com uma âncora com o estilo .util--no-style.

...
<figure class="mobile-nav__button">
  <img src="../img/icons/ico_autorizacoes.svg" alt="Autorizações">
  <figcaption>Autorizações</figcaption>
</figure>
...

Os modificadores do menu mobile

Botão ativo

O modificador .mobile-nav--active deve ser aplicado no elemento .mobile-nav__button que representa a página que está ativa no momento.

...
<a class="util--no-style">
  <figure class="mobile-nav__button mobile-nav--active">
    <img src="../img/icons/ico_autorizacoes.svg" alt="Autorizações">
    <figcaption>Autorizações</figcaption>
  </figure>
</a>
...

Aparecer em telas maiores

Para fazer o Menu Mobile funcionar também em dispositivos de tela grande, utilize a classe .mobile-nav--desktop.

<div class="mobile-nav mobile-nav--desktop">
  ...
</div>
