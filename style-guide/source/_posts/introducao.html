Começe a utilizar o Guia de Estilos Unimed agora mesmo

Quando definimos um Guia de Estilos de CSS para a Unimed, queremos unificar e uniformizar as nossas informações visuais digital. Temos aqui informações concretas sobre cores, tipografia, formas, componentes, exemplos de uso, tamanhos...

O estilo dos assets digitais da Unimed precisa parecer que foi desenvolvido de forma concisa por uma única pessoa, assim podemos manter a manutenção simples, garantindo projetos flexíveis e escaláveis.

Instalação

Estilos

A parte mais importante do Guia de Estilos é a folha de estilos CSS. Para começar a utilizar a folha de estilos padrão da Unimed, o primeiro passo é fazer o download dela.

Agora coloque essa folha dentro da pasta css da sua aplicação web e adicione o seguinte código no cabeçalho head das páginas do seu projeto:

<link rel="stylesheet" href="/css/style.css">

Pronto! É assim simples e você já pode começar a utilizar todos os estilos que o Guia de Estilos lhe proporciona.

Dica avançada

Você pode pegar os arquivos SCSS que montam a folha de estilo e montar uma folha própria, apenas com os elementos que interessar acessando o repositório desse projeto.

Baixar folha de estilos (.css)

Famílias tipográficas

No Guia de Estilos utilizamos como base da nossa estrutura tipográfica a família Unimed Sans. Também disponibilizamos acesso as famílias extras da Unimed, que são a Unimed Slab, a Unimed Serif e a Unimed Brush. Caso você não disponibilize elas no seu projeto, o Guia utilizará a família Trebuchet MS.

Para ter acesso as famílias oficiais no seu projeto primeiro faça o download delas. O segundo passo é adicionar os arquivos descompactados em uma pasta chamada typography que você criará dentro do diretório do seu projeto.

Agora você está pronto para utilizar as fontes junto com o a folha de estilos!

Primeiros passos

Para entender melhor como funcionam as classes e as nomenclaturas desse projeto leia a seção sobre a metodologia BEM.

Esse projeto segue as regras definidas pela Central da Marca Unimed, é importante estar familiarizado com ela para fazer aplicações em nome da Unimed.

O primeiro passo conhecendo esse Guia de Estilos é navegar pelas regras de Design e Layout. As principais leituras técnicas sobre o assunto podem ser encontradas nas seções de Tipografia e de Ritmo Vertical.
