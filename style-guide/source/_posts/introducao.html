Começe a utilizar o Guia de Estilos Unimed agora mesmo

Quando definimos um Guia de Estilos de CSS para a Unimed, queremos unificar e uniformizar as nossas informações visuais digital. Temos aqui informações concretas sobrwe cores, tipografia, formas, componentes, exemplos de uso, tamanhos...

O estilo dos assets digitais da Unimed precisar parecer que foi desenvolvido de forma concisa por uma única pessoa, assim podemos manter a manutenção simples, garantindo projetos flexíveis e escaláveis.

Instalação

Para começar a utilizar a folha de estilos padrão da Unimed, o primeiro passo é fazer o download dela.

Agora coloque essa folha dentro da pasta css do seu projeto HTML e adicione o seguinte código no cabeçalho (head) das páginas do seu projeto:

<link rel="stylesheet" href="/css/style.css">

Pronto! É assim simples e você já pode começar a utilizar tudo que o Guia de Estilos lhe proporciona.

Dica avançada

Você pode pegar os arquivos SCSS que montam a folha de estilo padrão e as famílias tipográficas no repositório desse projeto.

Primeiros passos

Para entender melhor como funcionam as classes e as nomenclaturas desse projeto leia a seção sobre a metodologia BEM.

Esse projeto segue as regras definidas pela Central da Marca Unimed, é importante estar familiarizado com ela para fazer aplicações em nome da Unimed.

O primeiro passo conhecendo esse Guia de Estilos é navegar pelas regras de Layout, como por exemplo as presentes na seção de Tipografia.
