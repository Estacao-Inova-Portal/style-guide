Usamos JavaScript para tornar alguns dos nossos componentes interativos

Para conseguir executar todas as interações do Guia de Estilos você deve utilizar nossa folha de scripts controllers.js e colocar ela na pasta /js do diretório da sua aplicação.

Baixar scripts (.js)

Troca de classes

A função toggleClass() é a responsável por fazer a alternância de classes. Ela é muito útil na hora de mostrar/esconder elementos, como um menu por exemplo.

Como usar

Ela precisa de dois parâmetros para funcionar, primeiro o id do elemento que vai sofrer a alternância e depois a classe CSS que será alternada.

Clique no botão a seguir para entender o seu funcionamento.

Troco de cor 
<a 
  onclick="toggleClass('exampleButton','btn--secondary-color-purple-dark')"
  id="exampleButton"
  class="btn btn--secondary-color-purple-dark">
Troco de cor
</a>
